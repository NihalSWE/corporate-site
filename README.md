# 🏢 Corporate Info Site

A professional Django-based corporate website.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/NihalSWE/corporate-site.git
cd corporate-site/info_site
```

### 2. Create Virtual Environment
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Apply Migrations
```bash
python manage.py migrate
```

### 5. Create Superuser
```bash
python manage.py createsuperuser
```

### 6. Run Server
```bash
python manage.py runserver
```
